## tuple

**有序且不可变的不同数据类型的集合**
### 创建元组
```py
empty_tuple =()
empty_tuple = tuple()
tpl = (item1,item2....)
```
### 元组长度
```py
len(tpl)
```
### 获取元组的项
```py
 tpl[index]
 tpl[para1:para2:para3]

```
### 将元组更改为列表
```py
lst = list(tpl)
```
### 检查元组中的项
item in tpl

### 连接元组
tpl1+tpl2

### 删除元组
del tpl#只可以全部删除
