import panda as pd

处理缺失值
```py
inputs,outputs = data.iloc[:,0:2],data.iloc[:,2]#iloc函数可以分割数据
inputs = inputs.fillna(inputs.mean())#用平均值填补空白,连续值缺失转化
inputs = pd.get_dummies(inputs, dummy_na=True)#离散值缺失转化
print(inputs)
```
