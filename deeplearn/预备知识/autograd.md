### 自动微分
系统根据构建好的模型会建立一个计算图，计算图的点存储参数，运算符号，中间结果，边表示数据的流向。计算机会记录某个张量上的操作，也就是一个变量时经过什么操作变成另一个变量：y=x^2
```py
x.requires_grad_(True)#需要追踪X的所有运算
=x =torch.arange(a,requires_grad=True)
x.grad#储存梯度的容器，初始为none，只有定义了以x为叶子的计算图，计算了某标量，调用backward时会填充梯度
x.grad.zero_()#在默认条件下会累计梯度，需要清除之前的值
y = x.sum()
y.backward()
x.grad
```
对于非标量的反向传播时，试图计算一批训练样本中每个组成部分的损失函数的导数。对于非标量不可以使用backward方法。如果要，需要提供gradient参数。gradient指的是权重向量，表示y的每一个分量对于向量x求导之后的结果进行加权求和，最终成为一个标量
```py
# 对非标量调用backward需要传入一个gradient参数，该参数指定微分函数关于self的梯度。
# 本例只想求偏导数的和，所以传递一个1的梯度是合适的
x.grad.zero_()
y = x * x
# 等价于y.backward(torch.ones(len(x)))##这里的torch.ones(len(y))指的就是权重函数
y.sum().backward()
x.grad
```
分离计算：对于复合函数求导法则的实现
我们有一个二元函数 z = f(u, x) = u * x，而 u 本来是 x 的函数 u = g(x)。
如果我们想求 全导数 dz/dx，根据链式法则：
dz/dx = ∂z/∂u * du/dx + ∂z/∂x
我们只关心z对于第二个变量的偏导数，而不要y参与
```py
x.grad.zero_()
y = x * x
u = y.detach()#u被赋予了y的值，但是丢弃如何计算y的信息
z = u * x

z.sum().backward()
x.grad == u
```
